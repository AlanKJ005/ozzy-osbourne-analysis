# Ozzy Osbourne Career Analysis Dashboard

## Overview
A Power BI dashboard analyzing Ozzy Osbourne's discography from 1981 to 2022, 
examining musical evolution across 12 albums and 137 tracks using Spotify audio features.

## Problem Statement
Ozzy Osbourne's albums were released over different periods, reflecting changes 
in musical style and album structure. This project uses Power BI to analyze and 
visualize these patterns chronologically.

## Tools Used
- Power BI Desktop
- Power Query (Data Cleaning)
- DAX (Measures)

## Dataset
- 137 tracks across 12 albums (1981-2022)
- Spotify audio features: Energy, Danceability, Valence, Tempo, Popularity

## Dashboard Pages
1. **Album Overview** - Track counts, duration trends, album summary
2. **Audio Characteristics Over Time** - Energy, Danceability, Valence, Tempo trends
3. **Track Deep Dive** - Top tracks, popularity analysis, Energy vs Popularity
4. **Era Analysis** - Early, Peak and Later era comparisons

## Key Insights
- Energy levels peaked during the Early Era (1981-1988)
- Danceability increased significantly in the Later Era (2001-2022)
- No More Tears (1991) remains his most popular album
- 67.88% of tracks are in Major key despite the dark image
